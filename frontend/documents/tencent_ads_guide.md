# 腾讯广告数据导入指南

> **提示**: 本指南将帮助您从腾讯广告投放平台导出数据并导入到省心投 BI 系统。

---

## 📋 数据说明

**数据类型**: 腾讯广告投放数据
**数据级别**: 账号级别
**导入格式**: .xlsx, .xls, .csv
**数据来源**: [腾讯广告ADQ平台](https://ad.qq.com/cm/home)

---

## 🔍 如何获取数据

### 步骤 1: 登录腾讯广告投放平台

1. 访问[腾讯广告 - 客户工作台](https://ad.qq.com/cm/home)
2. 使用您的账号登录

### 步骤 2: 进入报表中心

1. 点击顶部导航菜单的 **"分析"**
2. 选择左侧导航栏的 **"基础报表"**
3. 选择**账户分析**

### 步骤 3: 设置数据导出条件

1. **选择报表类型**: 广告报表
2. **选择日期范围**: 选择需要导出的日期范围
3. **选择维度**: 时间维度：分日
4. **选择指标**: 必须包含以下指标
   - ✓ 曝光量
   - ✓ 点击量
   - ✓ 点击用户数
   - ✓ 消耗（花费）
![[Pasted image 20260130102229.png]]

### 步骤 4: 导出数据

1. 点击 **"导出"** 按钮
2. 选择导出格式：**Excel** 或 **CSV**
3. 下载文件到本地

## ✅ 数据格式要求

### 必需字段（6个）

| 字段名 | 说明 | 格式要求 | 示例 |
|--------|------|----------|------|
| 日期 | 投放日期 | YYYY-MM-DD | 2025-01-15 |
| 账户ID | 广告账户ID | 文本 | 50252079 |
| 花费/消耗 | 投放花费 | 数值，保留2位小数 | 1234.56 |
| 曝光量/曝光次数 | 曝光次数 | 整数 | 100000 |
| 点击量/点击次数 | 点击次数 | 整数 | 5000 |
| 点击用户/点击人数 | 点击用户数 | 整数 | 3000 |

### 字段映射说明

系统支持以下列名变体（自动识别）：

| 标准字段 | 支持的列名 |
|---------|-----------|
| 日期 | 日期, 时间, 时间-天 |
| 账户ID | 账户ID, 账户, 账号 |
| 花费 | 花费, 消耗, 消费 |
| 曝光量 | 曝光量, 曝光次数, 曝光 |
| 点击量 | 点击量, 点击次数, 点击 |
| 点击用户 | 点击用户, 点击人数, 点击用户数 |

## ⚠️ 注意事项

### 数据要求

1. **日期格式**: 必须为 YYYY-MM-DD 格式（如：2025-01-15）
2. **数值格式**:
   - 花费可以是小数（如：1234.56）
   - 曝光、点击、点击用户数必须为整数
   - 支持千分位格式（如：1,234.56）
3. **必填字段**: 日期、账户ID、花费、曝光量、点击量、点击用户数

### 导入模式

- **覆盖模式**: 日期 + 账户ID 相同时，更新已有数据
- **数据去重**: 系统自动处理重复数据，保留最新导入的记录

### 常见问题

**Q: 导入后数据不显示？**
- 检查日期范围是否在筛选范围内
- 确认账号是否有代理商映射（可在账号管理中配置）
- 刷新页面重新加载

**Q: 提示"缺少必需字段"？**
- 确认Excel/CSV文件包含所有必需列
- 检查列名是否正确（参考字段映射表）

**Q: 数值格式错误？**
- 确保花费为数字格式（非文本）
- 检查是否有特殊字符（如货币符号）

## 📥 导入步骤

1. 在数据导入页面，选择 **"腾讯广告"** 数据类型
2. 点击上传区域或拖拽文件到上传框
3. 选择导出的文件（.xlsx, .xls 或 .csv）
4. 确认文件信息无误
5. 点击 **"开始导入"** 按钮
6. 等待导入完成（查看导入进度）
7. 导入完成后，前往 **账号管理** 补充账号映射信息

---

## 🔗 后续操作

### 账号映射配置

导入成功后，建议立即配置账号代理商映射：

1. 进入 **配置管理** → **账号管理**
2. 找到新导入的腾讯账号
3. 补充以下信息：
   - 账号名称
   - 代理商名称
   - 业务模式（直播/信息流/搜索）
4. 保存配置

### 数据验证

导入完成后，可在以下报表中验证数据：

- **数据概览**: 查看整体花费趋势
- **厂商分析**: 按代理商维度查看腾讯数据
- **线索明细**: 查看腾讯平台的转化数据
