## 🔍 如何获取数据

### 步骤 1: 登录小红书星火平台

1. 访问 [小红书星火平台](https://mcc.xiaohongshu.com/)
2. 使用您的账号登录

### 步骤 2: 进入报表中心

1. 点击顶部导航菜单的 **"数据"**
2. 左侧菜单栏选择 **"工具-自助分析工具"**
3. 数据集选择-创意数据集（即从笔记素材维度统计数据的数据集）

### 步骤 3: 设置数据导出条件
![[Pasted image 20260130103043.png]]
1. **选择报表类型**: 广告报表
2. **选择日期范围**: 选择需要导出的日期范围
3. **选择维度**: 按代理+笔记+按天维度
4. **选择指标**: 必须包含以下指标
   - ✓ 日期
   - ✓ 笔记ID
   - ✓ 总消耗
   - ✓ 总展现
   - ✓ 总点击
   - ✓ 点赞数
   - ✓ 收藏数
   - ✓ 评论数
   - ✓ 分享数
   - ✓ 关注数
   - ✓ 私信进线数

### 步骤 4: 导出数据

1. 点击 **"导出"** 按钮
2. 选择导出格式：**Excel** 或 **CSV**
3. 下载文件到本地

## 🎯 数据用途

笔记日级投放数据用于：
- 记录每个笔记的投放消耗和效果
- 记录投放带来的互动数据（点赞、评论、收藏等）
- 与笔记业务数据对比，拆分自然流量
- 支持按笔记、代理商等维度分析

### 自然流量计算

```
自然流量 = 总业务数据 - 投放数据

自然曝光 = 笔记业务数据.total_impressions - 笔记投放数据.impressions
自然互动 = 笔记业务数据.total_interactions - 笔记投放数据.total_interactions
```

## ⚠️ 注意事项

### 数据要求

1. **日期格式**: 必须为 YYYY-MM-DD 格式（如：2025-01-15）
2. **数值格式**:
   - 消耗可以是小数（如：1234.56）
   - 其他指标必须为整数
   - 支持千分位格式（如：1,234.56）
3. **必填字段**:
   - ✅ 日期
   - ✅ 笔记ID
   - ✅ 消耗流水
   - ✅ 曝光量
   - ✅ 点击量
   - 其他字段可选（推荐填写）

### 数据唯一性

- **唯一约束**: 日期 + 笔记ID
- **覆盖模式**: 相同组合的数据会被更新

### 代理商识别

如果导出文件包含账户ID字段（广告主账户ID、子账户ID），系统会自动关联代理商：
- 通过子账户ID关联 `account_agency_mapping` 表
- 自动填充代理商信息

### 常见问题

**Q: 如何区分投放数据和自然流量数据？**
- 本数据类型：投放数据（付费流量）
- 笔记日级业务数据：总业务数据（含自然流量）
- 自然流量 = 总业务数据 - 投放数据

**Q: 互动指标缺失可以吗？**
- 可以，互动指标为可选字段
- 但建议完整填写，以便拆分自然流量

**Q: 如何配置代理商映射？**
- 在"账号管理"中配置小红书账号映射
- 使用广告主账户ID和子账户ID进行映射
