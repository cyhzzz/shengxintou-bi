## 🎯 数据用途

笔记日级业务数据用于：
- 记录笔记的总业务数据（含自然流量）
- 与投放数据对比，拆分自然流量
- 分析笔记的运营效果
- 评估创作者绩效

### 自然流量计算

```
自然流量 = 总业务数据 - 投放数据

自然曝光 = 笔记业务数据.total_impressions - 笔记投放数据.impressions
自然互动 = 笔记业务数据.total_interactions - 笔记投放数据.total_interactions
```

### 投放ROI计算

```
投放ROI = 投放互动 / 投放花费
自然流量ROI = 自然互动 / 0（自然流量无花费）
```

## ⚠️ 注意事项

### 数据要求

1. **日期格式**: 必须为 YYYY-MM-DD 格式（如：2025-01-15）
2. **数值格式**:
   - 所有数值字段支持整数和小数
   - 支持千分位格式（如：1,234.56）
3. **必填字段**:
   - ✅ 数据日期
   - ✅ 笔记ID
   - ✅ 全部曝光量
   - ✅ 全部阅读量
   - ✅ 全部互动量
   - 其他字段可选

### 数据唯一性

- **唯一约束**: 数据日期 + 笔记ID
- **覆盖模式**: 相同组合的数据会被更新

### 数据来源说明

- **投放平台**: 获取笔记投放数据（消耗、展现、点击等）
- **专业号平台**: 获取笔记业务数据（曝光、阅读、互动等）

两者需要分别导出并导入到不同数据类型：
- 投放数据 → **笔记日级投放数据**
- 业务数据 → **笔记日级业务数据**（本类型）

### 常见问题

**Q: 笔记日级业务数据和投放数据有什么区别？**
- 业务数据：总数据（包含自然流量 + 投放流量）
- 投放数据：仅投放带来的数据（付费流量）

**Q: 为什么需要同时导入两种数据？**
- 通过对比可以拆分自然流量和付费流量
- 评估投放效果和自然效果
- 计算真实的投放ROI

**Q: 互动量缺失可以吗？**
- 可以，互动量为可选字段
- 但建议完整填写，以便分析笔记效果

## 💡 最佳实践

### 1. 数据导入顺序

```
1. 导入笔记列表（基础信息）
2. 导入笔记日级投放数据（投放数据）
3. 导入笔记日级业务数据（业务数据）
4. 运行聚合脚本（合并数据）
```

### 2. 自然流量分析

- 同时导入投放数据和业务数据
- 聚合脚本自动计算自然流量
- 在运营分析报表中查看对比

### 3. 数据更新频率

- **建议频率**: 每天一次
- **更新时间**: 上午10点后（平台数据已更新）
- **日期范围**: 建议单次导入不超过31天
