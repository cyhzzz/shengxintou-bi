# 省心投 BI - 使用说明

## 快速启动

双击运行：`省心投启动器.exe`

启动器会自动：
1. 检查Python环境
2. 检查依赖包
3. 启动Flask服务器
4. 打开浏览器

访问地址：http://127.0.0.1:5000

---

## 目录结构说明

```
开发代码/
├── 省心投启动器.exe        # 主启动器（双击运行）
├── python-3.9-embed/       # 便携Python环境（无需安装Python）
├── lib/                    # Python依赖包
├── app.py                  # 应用入口
├── config.py               # 配置文件
├── backend/                # 后端代码
├── frontend/               # 前端代码
├── database/               # 数据库文件
├── uploads/                # 文件上传目录
└── logs/                   # 日志文件
```

## 常见问题

### Q1: 启动器无反应？
**解决**：
1. 检查是否有杀毒软件拦截
2. 右键选择"以管理员身份运行"

### Q2: 浏览器无法打开？
**解决**：
手动访问 http://127.0.0.1:5000

### Q3: 端口5000被占用？
**解决**：
1. 按Ctrl+C停止服务器
2. 关闭占用端口的程序
3. 重新启动

---

## 停止服务

在启动器窗口按 `Ctrl+C` 即可停止服务器。
